\# Ziply    
\*\*AI-Assisted Website Generation for Nonprofit Organizations\*\*

\#\# Overview  
Ziply is a software-as-a-service (SaaS) platform that enables nonprofit organizations to generate professional, customizable websites without requiring any coding experience. Through a guided, no-code workflow, users select a nonprofit-focused template, enter organization details, and receive a complete, deployable website as a downloadable ZIP file.

This project is being developed as part of a \*\*Computer Science Capstone\*\* and focuses on building a real-world, production-style system using modern full-stack and cloud technologies.

\---

\#\# Motivation & Social Impact  
Many nonprofit organizations operate with limited budgets and lack access to dedicated technical support. Professional web development services are often cost-prohibitive, while free website builders can limit ownership, customization, or credibility.

Ziply addresses this gap by providing:  
\- Affordable access to a professional web presence  
\- Full ownership of generated websites (no vendor lock-in)  
\- Templates designed specifically for nonprofit needs such as donations, volunteer sign-ups, and program visibility

By lowering the technical and financial barriers to building a web presence, Ziply supports organizations that serve communities and create positive social impact.
